## 🚀 Dinamik Dashboard & Mesaj Yönetim Sistemi

## 📌 Proje Özellikleri
- 🔐 Kullanıcı kayıt ve giriş sistemi (ASP.NET Identity)
- 📬 Mesajlaşma sistemi (Gelen Kutusu / Giden Kutusu)
- ⭐ Mesaj yıldızlama (önemli mesajları işaretleme)
- 📝 Taslak mesaj oluşturma
- 📊 Dinamik dashboard istatistikleri
- 📅 Günlük / Haftalık / Aylık mesaj analizleri
- 📌 Okunan & Okunmamış mesaj takibi
- 🧩 ComponentPartial yapısı ile modüler mimari
- 🎨 Modern ve responsive dashboard tasarımı


## 📈 Dashboard İçeriği

Dashboard ekranında kullanıcıya özel olarak aşağıdaki veriler sunulmaktadır:

### ⏱️ Zaman Bazlı Veriler
- Günlük Mesaj
- Haftalık Mesaj
- Aylık Mesaj
- Toplam Mesaj
### 📬 Durum Bazlı Veriler
- Okunan Mesaj
- Okunmamış Mesaj
- Yıldızlı Mesaj
- Taslak Mesaj

## 🛠️ Kullanılan Teknolojiler
- Backend: ASP.NET Core MVC, C#
- Frontend: Razor View Engine, HTML, CSS, Bootstrap
- Database: SQL Server
- ORM: Entity Framework Core
- Grafik: Chart.js
- Mimari: Component-Based yapı (Partial & ViewComponent)
